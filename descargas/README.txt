CARPETA DE DESCARGAS
====================

Coloca aquí todos los documentos y ensayos que desees que estén disponibles para descargar.

Formatos soportados:
- PDF (.pdf)
- Word (.doc, .docx)
- Excel (.xls, .xlsx)
- PowerPoint (.ppt, .pptx)
- Texto (.txt)
- Imágenes (.jpg, .png, .gif)

Los archivos aparecerán automáticamente en la página de descargas del sistema.
